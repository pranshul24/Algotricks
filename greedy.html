Solution

Let us say we have a function f that find the number of ab subsequences in the string . Let '+' be concatenate operator . Now let C and D be 2 strings then :
f(C+D) gives the number of subsequences ab in the string C+D
f(D+C) gives the number of subsequences ab in the string D+C
f(D+C) = E + F + Da * Cb
f(C+D) = F + E + Ca * Db
So now we keep C before D only when f(C+D) is greater than f(D+C)
So , Ca * Db > Da * Cb
Ca / Cb > Da / Db
So, from this greedy stratergy we conclude that we conclude that the sort is transitive if we use a comparator function in the code .
